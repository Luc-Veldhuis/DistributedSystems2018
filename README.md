# DistributedSystems2018 - Team 6

Asynchronous Job Runner

## Getting Started

Use Intellij for the SDE

Download Gradle

### Installing

Gradle 
* [Gradle](https://gradle.org/install/) 


## Build & Start Project

Go to the root folder of the project and run the following commands
```
gradle build
```

```
gradle start
```
